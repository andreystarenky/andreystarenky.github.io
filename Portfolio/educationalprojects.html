ICS4U0 Culminating Assignment - Overcoming Adversity Game

This Java project is the final assignment in AP grade 12 Computer Science. The assignment is to create an Overcoming Adversity Game.
The task is to create a game to teach and help people with somethingin their lives.
We chose to focus on teaching people etiquette through 3 levels: A lesson with a quiz, a recognition test, and an interactive role-playing game.
The project incorporates the use of complex Object Oriented Programming concepts, as well as FileIO, Searching and Sorting, JavaFX GUI, and Memory Management.

ICS3U Culminating Assignment - Snake Game

This Java project is the final assignment in grade 11 Computer Science. The assignment is to create a Snake Game.
The program allows the user to control a snake that grows by eating pellets, but dies when colliding with itself or the border.
The project incorporates the use of multiple classes, threads, and collison detection.
