Java Object Oriented Programming Demonstration


This is a basic demonstration of Oject Oiented Pogramming in Java.
It consists of a super class PeripheralDevice which has basic characteristics and behaviours such as connectionType and disconnect()
and a subclass OutputDevice which extends PeripheralDevice and has it's own characteristics and behaviours such as resolutionWidth and cropResolution().
There is also a RunMain.java class which creates Instances of the classes and demonstrates the behaviours of the Objects.
