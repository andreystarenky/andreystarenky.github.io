Arrays
 Arrays in Java are data structures that can store multiple elements of the same data type.
When initializing an array, you can either specify the number of elements or all the elements directly.
To initialize with the number of elements: Type[] arrayName = new Type[# of elements]; Ex. Integer[] arr = new Integer[10];
To initialize directly with the elements: Type[] arrayName = {element value 1, element value 2, etc}; Ex. String[] arr = {"this", "is", "an", "array"};
Arrays have a length field, which represents the number of elements in the array.
To access a value in the array, you need to specify the index of the element. The first element has index 0, and the last has arr.length - 1; Ex. System.out.print(arr[2]);
When dealing with Arrays of Numeric or String values, you can use Arrays.sort(arr); to sort the array from least to greatest value
You may also have 2D Arrays, arrays that consist of other arrays. Then, to access for instance the 2nd element in the 3rd array, you would do matrix[2][1]

Arraylists
 Arraylists are similar to Arrays in the sense that they are data structures that can store multiple elements of the same type,
however they have variable lengths and cannot be of the primitive type.
Arraylists are initialized as such: Arraylist <type> arrList = new Arraylist<type>();
By default, the size of the ArrayList is set to 10, and it increases as needed.
To add elements to an ArrayList, use arrList.add(value) or arrList.add(index,value)
To remove elements from an ArrayList, use arrList.remove(value) or arrList.add(index)
