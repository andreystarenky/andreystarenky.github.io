Recursion


 Recursion is the process of reaching a solution by relying on the result of smaller solutions. Recursion is typically achieved by one or more methods calling themselves.
For recursive solutions to reach an end and an not be called infinitely, they will need a base case. A base case returns a value without making any recursive calls.
Linear Recursion occurs when a method calls itself recursively.
Mutual Recursion occurs when 2 methods recursively call eachother.
Tail Recursion is used when no more computing is done after the recursive call.
A real life example of recursion is a camera pointing at a TV displaying the output of the camera. This will cause an infinite display of the TV.
A programmatic example of recursion is finding a factorial value. The base case is to reuturn 2 when the number is 2, otherwise return the recursive call of the factorial
method with the absolute value of the number - 1.
